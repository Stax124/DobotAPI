<!-- GETTING STARTED -->

## Getting Started

You will need the dobot of course, and system runing Linux (tested on x86_64)

### Prerequisites

- python (3.10 tested, might run on older versions)
  ```sh
  sudo apt install python3 pip3
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Stax124/DobotAPI
   ```
2. Run the app
   ```sh
   python3 example.py
   ```
